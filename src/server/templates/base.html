single file → "chop it up!": enter a url or upload a single file, set start/end (optional), select patch type, and hit "chop it up!". drum patches are automatically split into splices based on waveform transients. use # splices to override the automatic detection with a fixed number. the chop it up! button is only relevant in this single-file mode.

multi-upload → fully automatic (op-z extensions): select as many individual sample files as you want — no limit. no chopping needed: each file is already one sample. the tool automatically groups them into as many .aif drum kits as needed, respecting both limits the op-z imposes: max 24 splices and max 12 s total per kit. so 60 samples become e.g. 3–4 kits, each ready to drop onto the op-z.

synth batch: select multiple audio files in synth mode — each one is converted to its own .aif synth patch. the op-z synth slot holds up to 5.75 s per sample. all files are bundled into a single zip for download, ready to drop into op-z synth slots.

compress mode (drum & synth): for samples longer than 1 s, compress halves the sample rate (+1 octave). on the op-z, set the pitch parameter to −1 oct → original pitch is restored and you effectively get double the playback length out of each slot: drum kits: up to 24 s effective playback (from 12 s slot) · synth patches: up to ~11.5 s effective playback (from 5.75 s slot). useful for longer loops, pads, or textures that would otherwise be cut short.

drum examples: drumbeat (instagram), poetry (library of congress), spoken word (youtube), elektron digitone fm drums (gumroad, free pack → download & upload via multi-upload).

synth examples: cello, piano, didgeridoo, epic whistle.
